# Data

Raw data is not committed to this repo (619MB, exceeds GitHub's 100MB per-file
limit, and redistribution of the raw Kaggle files isn't covered by any license
here). Everything is reproducible from the scripts below.

## Lending Club loans (`data/raw/`)

Source: [Kaggle — wordsforthewise/lending-club](https://www.kaggle.com/datasets/wordsforthewise/lending-club)
(accepted + rejected loans, 2007–2018Q4).

```bash
# 1. Get a Kaggle API token: https://www.kaggle.com/settings -> API -> Create New Token
#    Save as ~/.kaggle/kaggle.json (or %USERPROFILE%\.kaggle\kaggle.json on Windows)
pip install kaggle
python -m credit_risk.data.download
```

Populates:
- `data/raw/accepted_2007_to_2018Q4.csv.gz`
- `data/raw/rejected_2007_to_2018Q4.csv.gz`

## Macroeconomic series (`data/processed/macro_quarterly.csv`)

Source: [FRED](https://fred.stlouisfed.org/) (St. Louis Fed) — `UNRATE`, `GDP`,
`CPIAUCSL`, `FEDFUNDS`, `CSUSHPISA`, quarterly, 2007Q1–2018Q4. This file *is*
committed (small, public, no PII).

```bash
# Optional: set FRED_API_KEY to pull live data; falls back to a bundled
# high-fidelity historical snapshot if no key/network is available.
python -m credit_risk.data.download_macro
```

## Interim / processed pipeline outputs

Generated by the pipeline (`make pipeline` / `python -m credit_risk.pipeline`)
and gitignored — regenerate locally after downloading the raw data above.
